# 系统架构设计

## 总体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        Web 前端界面                             │
│  (ros_car_web_demo - Vue.js)                                    │
└─────────────────────────┬───────────────────────────────────────┘
                          │ WebSocket/HTTP
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                        ROS 主节点                               │
│  (car_control/ros/ - Python/C++)                                │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 串口通信
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                        STM32 控制器                             │
│  (car_control/stm32/ - C)                                       │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 电机控制信号
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                        硬件执行层                               │
│  (电机、传感器、摄像头)                                          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                   图像处理子系统                                │
│  (optimize_stitching/ - Python/OpenCV)                         │
│  ├── 视频读取 → 帧采样 → 光流估计 → 图像拼接 → 图像增强 → 输出   │
└─────────────────────────────────────────────────────────────────┘
```

## 模块交互

### 车辆控制流程

```
遥控器(SBUS) → sbus_serial_node → SBUS消息 → straight_controller → 串口指令 → STM32 → 电机
```

### 图像处理流程

```
视频文件 → VideoReader → AdaptiveSampler → ContentDetector → OpticalFlow → Stitcher → OutputWriter
```

## 数据流

| 数据类型 | 来源 | 目标 | 传输方式 |
|---------|------|------|---------|
| SBUS控制信号 | 遥控器 | ROS节点 | 串口 |
| 控制指令 | ROS节点 | STM32 | 串口 |
| 状态反馈 | STM32 | ROS节点 | 串口 |
| 实时视频 | 摄像头 | Web前端 | WebSocket |
| 拼接结果 | Stitcher | 输出文件 | 磁盘 |

## 关键技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue.js | Web控制界面 |
| 通信 | ROS | 机器人操作系统 |
| 控制 | C/C++/Python | 控制逻辑实现 |
| 固件 | STM32 HAL | 嵌入式开发 |
| 图像处理 | OpenCV/PyTorch | 图像拼接与特征匹配 |
