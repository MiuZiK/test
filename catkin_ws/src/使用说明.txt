2.创建设备STM32运动底盘串口的规则
打开文件所在路径：cd turn_on_wheeltec_robot\scripts
执行创建规则的文件：sudo ./wheeltec_udev.sh
拔插STM32运动底盘串口，使规则生效

创建规则相关教程：【资料包\3.ROS开发手册\4.ROS开发教程.pdf】

3.根据编译提示手动安装相关依赖包
ROS相关依赖包：sudo apt-get install ros-melodic-xxx*
系统相关依赖包：sudo apt-get install xxx*
带msgs的都是ROS相关依赖，其它需要自行判断例如提示没有uuid_msgs，就运行：sudo apt-get install ros-melodic-uuid*
提示缺少serialConfig，就安装sudo apt-get install ros-melodic-serial（其它ROS版本如果没有这个功能包，我们不负责）

4.自行创建 ubuntu 环境编译/运行功能报错 
首先自行安装 rosdep 后，在工作空间下运行以下命令安装绝大部分功能包依赖： rosdep install --from-paths src --ignore-src -r -y 
编译提示缺的功能包可以通过命令安装：sudo apt-get install ros-melodic-功能包名称 
部分功能适配观看教程【资料包/3.ROS 开发手册\相关功能安装教程】
